# Project Brief: Kids Menu Ordering App

## Overview
A kid-friendly digital menu and ordering application that allows children to browse and select meals from a curated menu, with parent oversight and order management.

## Core Requirements
- **Kid-Friendly UI**: Colorful, engaging interface designed for children
- **Profile Selection**: Multiple child profiles with avatars
- **Meal Ordering**: Browse categories (mains, sides, drinks, desserts, snacks) and select items
- **Order Submission**: Send orders to kitchen via Google Sheets integration
- **Parent Dashboard**: PIN-protected admin area for managing orders and menu items
- **Order Tracking**: Real-time order queue with status management

## Goals
1. Make meal ordering fun and engaging for kids
2. Simplify the ordering process with a guided flow (mains → sides → drinks)
3. Provide parents with full control over menu items and order management
4. Maintain order history and tracking

## Scope
- Single-page React application with routing
- V3 Duolingo-style design (current default)
- Supabase integration for order storage
- Google Sheets integration as backup
- PWA capabilities for mobile use
